## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with phenomenon documentation
- api.json following Wyrdness schema v1.0.0
- SOURCES.md bibliography
- Issue templates for sighting reports, corrections, and source additions
- Pull request template

### Data Included
- Basic phenomenon metadata and classification
- Physical characteristics and abilities
- Geographic distribution (Pine Barrens, NJ)
- Historical timeline (1735-present)
- 4 notable sightings including 1909 Phenomenal Week mass sighting
- Evidence catalog (physical, photographic)
- Theories (paranormal, cryptozoological, skeptical including sandhill crane hypothesis)
- Cultural significance and New Jersey folklore
- Research organizations and notable researchers
- Mother Leeds legend documentation
